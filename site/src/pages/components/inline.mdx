# Inline

The Stack component distributes its children vertically down a page with equal spacing in between.

---

### Usage

<LiveCode>
  <LiveCode.Content
    title="Default Inline Spacing"
    code={`
      <Inline>
        <Box p="300" bg="gray.200">Box</Box>
        <Box p="300" bg="gray.200">Box</Box>
        <Box p="300" bg="gray.200">Box</Box>
      </Inline>
    `}
  />
  <LiveCode.Content
    title="Right Aligned"
    code={`
      <Inline align="right" space="100">
        <Box p="300" bg="gray.200">Box</Box>
        <Box p="300" bg="gray.200">Box</Box>
        <Box p="300" bg="gray.200">Box</Box>
      </Inline>
    `}
  />
  <LiveCode.Content
    title="Bottom and Right Aligned"
    code={`
      <Inline align="right" alignY="bottom" space="100">
        <Box p="300" bg="gray.200">Box</Box>
        <Box p="500" bg="gray.200">Box</Box>
        <Box p="300" bg="gray.200">Box</Box>
      </Inline>
    `}
  />
</LiveCode>

---

### System Props

None

---

### Stack API

<Prop
  name="align"
  type="string | array"
  values={['"center"', '"left"', '"right"', 'null']}
>
  Positions children horizontally. Also accepts responsive arrays.
</Prop>

<Prop
  name="alignY"
  type="string | array"
  values={['"center"', '"top"', '"bottom"', 'null']}
>
  Positions children vertically. Also accepts responsive arrays.
</Prop>

<Prop name="space" type="string | array">
  Sets the gutter space between children. Also accepts responsive arrays.
</Prop>
