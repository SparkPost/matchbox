### Getting Started with Matchbox

First install the component library with <ExternalLink tp="https://www.npmjs.com/package/@sparkpost/matchbox">NPM</ExternalLink>. Our components are styled with javascript via <ExternalLink to="https://styled-components.com/">styled-components</ExternalLink>, install this too.

```bash
npm install @sparkpost/matchbox @sparkpost/matchbox-icons styled-components --save
```

Optionally, if you plan on styling with Matchbox, install our [design token library](/design/overview).

```bash
npm install @sparkpost/design-tokens --save
```

Wrap your React app with our `ThemeProvider` component:

```javascript
import { ThemeProvider } from '@sparkpost/matchbox';

function App(props) {
  return <ThemeProvider>{props.children}</ThemeProvider>;
}
```

That's it!
