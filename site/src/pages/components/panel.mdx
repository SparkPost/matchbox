# Panel

A panel is a visual grouping of similar content or tasks in order create readable and clear experiences for SparkPost's users.

---

### Panel API

<Prop name="title" type="node">
  The title of the panel
</Prop>

<Prop name="className" type="string">
  Optional class name for the panel
</Prop>

<Prop name="accent" type="bool | string">
  orange, blue, red, yellow, green, purple, navy, gray <br />
  Renders an accent border on top of the panel
</Prop>

<Prop name="sectioned" type="bool">
  Whether or not the panel will have sections
</Prop>

<Prop name="actions" type="array">
  Lost of actions for the panel. Accepts objects with button props
</Prop>

<Prop name="children" type="node">
  The panel content
</Prop>

### Panel.Section API

<Prop name="actions" type="array">
  Lost of actions for the panel section. Accepts objects with button props
</Prop>

<Prop name="children" type="node">
  The section content
</Prop>

<Prop name="rootRef" type="function">
  Maps to Downshift's refKey set in getRootProps. refKey must be set to
  "rootRef"
</Prop>
