# Panel

A panel is a visual grouping of similar content or tasks in order create readable and clear experiences for SparkPost's users.

---

### Usage

<LiveCode>
  <LiveCode.Content
    title="With a Title"
    code={`

        <Panel.Header>Title</Panel.Header>
        <Panel.Section>Section Content</Panel.Section>
      </Panel>
    `}
  />
  <LiveCode.Content
    title="With an accent bar"

### Panel System Props

<SystemPropsTags
  propsList={['margin', 'padding', 'border', 'height', 'width']}
/>

### Panel API

<Prop
  name="accent"
  type="bool | string"
  values={['"orange"', '"blue"', '"red"', '"yellow"', '"green"', '"gray"']}
>
  Renders an accent border on top of the panel
</Prop>

<Prop name="children" type="node">
  The panel content
</Prop>

<Prop name="className" type="string">
  Optional class name for the panel wrapper
</Prop>

---

### Panel.Header System Props

None

### Panel.Header API

<Prop name="as" type="string" defaultValue='"h3"'>
  The wrapping element around text
</Prop>

<Prop name="children" type="node">
  Panel header content
</Prop>

<Prop name="className" type="string">
  Optional class name for the panel header wrapper
</Prop>

---

### Panel.SubHeader System Props

None

### Panel.SubHeader API

<Prop name="as" type="string" defaultValue='"h4"'>
  The wrapping element around text
</Prop>

<Prop name="children" type="node">
  Panel subheader content
</Prop>

<Prop name="className" type="string">
  Optional class name for the panel subheader wrapper
</Prop>

### Panel.Section System Props

<SystemPropsTags propsList={['padding']} />

### Panel.Section API

<Prop name="children" type="node">

<Prop name="className" type="string">
  Optional class name for the panel section wrapper
</Prop>

### Panel.Action System Props and API

This component is a light wrapper around the Button component that sets size and color. See <Link to='/components/button'>the Button component page</Link> for API documentation and usage.
