### New Components

##### Layout

The new Layout component is built to be used to render the two-column annotated layouts that Hibana uses.

- <ExternalLink to="https://matchbox-storybook.netlify.app/?path=/story/layout-layout--column-example">
    Layout stories
  </ExternalLink>

```jsx
<Layout>
  <Layout.Section annotated>
    <Layout.SectionTitle>Section Title</Layout.SectionTitle>
    Annotated Section Content
  </Layout.Section>
  <Layout.Section>Main Section Content</Layout.Section>
</Layout>
```

##### Composable Banner Components

The Banner component now includes one new component, `Banner.Action`. The `actions` and `action` props will continue to work, but have been deprecated.

- <ExternalLink to="https://matchbox-storybook.netlify.app/?path=/story/feedback-banner--banner-with-actions">
    New Banner stories
  </ExternalLink>
- <ExternalLink to="https://matchbox-storybook.netlify.app/?path=/story/feedback-banner--banner-with-deprecated-actions">
    Deprecated Banner stories
  </ExternalLink>

```jsx
<Banner status="info" title="Banner Title">
  <p>Banner Content</p>
  <Banner.Action>Action Content</Banner.Action>
</Banner>
```

##### Composable Panel Components

Panel now includes two new components, `Panel.Header`, `Panel.Action`. The deprecated component, `Panel.LEGACY`, is also available and is identical to the pre `4.1.0` Panel.

- <ExternalLink to="https://matchbox-storybook.netlify.app/?path=/story/layout-panel--with-a-header">
    New Panel stories
  </ExternalLink>
- <ExternalLink to="https://matchbox-storybook.netlify.app/?path=/story/layout-legacy-panel--with-a-title">
    Legacy Panel stories
  </ExternalLink>

```jsx
<Panel>
  <Panel.Header>
    Header Title
    <Panel.Action>Action</Panel.Action>
    <Panel.Action>Action</Panel.Action>
  </Panel.Header>
  <Panel.Section>
    Section Content
    <Panel.Action>View Details</Panel.Action>
  </Panel.Section>
</Panel>
```

##### Composable Modal Components

Modal has recieved an overhaul and aligns closer to the current `Drawer` prop interface. Modal now includes a `Panel` by default. The deprecated component, `Modal.LEGACY` is also available and is identical to the pre `4.1.0` Modal.

- <ExternalLink to="https://matchbox-storybook.netlify.app/?path=/story/overlays-modal--open">
    New Modal stories
  </ExternalLink>
- <ExternalLink to="https://matchbox-storybook.netlify.app/?path=/story/overlays-modal--legacy">
    Deprecated Modal stories
  </ExternalLink>

```jsx
<Modal closeOnEscape closeOnOutsideClick>
  <Modal.Header showCloseButton>Modal Title</Modal.Header>
  <Modal.Content>Modal Content</Modal.Content>
  <Modal.Footer>
    <Button>Primary Button</Button>
    <Button>Secondary Button</Button>
    <Button>Tertiary Button</Button>
  </Modal.Footer>
</Modal>
```

##### RadioCard and RadioCard.Group

`RadioCard` and `RadioCard.Group` are two new components that render radio inputs as Panels

### New Hooks

##### useResizeObserver

Gets the `rect` object of the referenced DOM element and updates it when it is resized.

- <ExternalLink to="https://matchbox-storybook.netlify.app/?path=/story/utility-useresizeobserver--example-usage">
    Example Story
  </ExternalLink>

```jsx
import { useResizeObserver } from '@sparkpost/matchbox';

function Component() {
  const [ref, entry] = useResizeObserver();
  console.log(entry.contentRect);
  return <div ref={ref} />;
}
```

##### useCopyToClipboard

SSR friendly hook that copies a string to the clipboard and provides copied state

- <ExternalLink to="https://matchbox-storybook.netlify.app/?path=/story/utility-usecopytoclipboard--example-usage">
    Example Story
  </ExternalLink>

```jsx
import { useCopyToClipboard } from '@sparkpost/matchbox';

function Component() {
  const { copy, copied } = useCopyToClipboard({ timeout: 2000 });

return (
  <Button onClick={() => copy('string')}>
    {copied ? 'String has been copied' : 'Copy to clipboard'}
  </Button>;
  )
};
```
